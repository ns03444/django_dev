About Me

Joining Fiserv in June 2021 as part of the Client WAN Engineering Organization, Nicholas is an active member of a DevOps teams delivering Network Automation through the latest tools and software in an agile fashion. Their mission is to change the way our Network Engineers support clients by automating repetitive, error-prone tasks from their daily operations. Our automation platform is continuously improving the way changes are deployed, the cost of operations, and the delivery time of IT services. Areas of specialization include:

  • Python Scripting with NETCONF, Netmiko, NAPALM, etc. to automate network device provisioning & configuration change executions.
  • Developing custom Ansible modules & plugins integrated with Python tools & REST APIs.
  • Contributing to collaborative projects using Git, Gitlab, CI/CD, formatting standards, and semantic versioning.
  • Testing containerized workflows on networking devices in a virtual lab environment with custom unit and integration testing
